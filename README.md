# GeoMIP_for_CMIP7_analysis_code
Plotting and processing code to produce Figures in Visioni et al. (2026 - forthcoming); _The Geoengineering Model Intercomparison Project (GeoMIP) contribution to CMIP7_. 

**Contents**
1. temp_fig3.ipynb -- author: Cindy Wang (cw2239@cornell.edu) -- produces Figure 2, the evolution of global-mean temperature, interhemispheric temperature gradient, and equator-to-pole temperature gradient, under SSP2-4.5 G6-1.5K-SAI, ARISE-SAI-1.5 and G6-1.5K-HiLLA. Notebook runs on Derecho HPC.
2. G6-1.5K-SAI_HiLLA_example.iynb -- author: Alistair Duffey (alistair@reflective.org) -- produces Figure 4, an overview of SO2 injection magnitudes, temperature evolution and zonal temperature responses under the G6-1.5K-SAI and -HiLLA scenarios. Notebook runs on Reflective Cloud Hub.
